Transparency is required in every business; when customers purchase anything from the seller then they may need to ask multiple questions. Hence, clear conversation is required so that buyers can make the purchase without any second thought. E-mails are the great way for formal conversation.

However, conversations through e-mails can be a bit slow, which can directly hamper the opportunity of sales. When thinking about a faster way for conversation, live chats are the best. With the help of Live chats, the sellers can easily convert the website visitor into the potential buyer.

Now, launch a live chat option in your Odoo Marketplace also. Odoo Marketplace Buyer Seller Live Chat module facilitates your sellers to chat with the customers without any hassle. All the conversation is maintained in the Odoo backend for future purposes. Moreover, a customer can chat with multiple sellers at a time.

Detailed Features List

Below is the detailed list of Feature for Marketplace Buyer Seller Live Chat

Allow Buyer Seller Communication

  • Offer Live Chat option between sellers and customers.
  • The module allows the buyers to directly contact sellers for any query or feedback.
  • The buyer can contact sellers from the product page or seller profile.

Offer Communication Flexibility

  • The customers/sellers can send emoticons in the live chats.
  • Customers can communicate with multiple sellers at a time and vice versa.

Manage the thread from front-end

  • Customers can check all the message threads from all sellers from front-end.

Better Tracking Management

  • The Super admin can track conversation from all sellers on the marketplace.
  • Moreover, as an admin you can also check details of each message from the backend.

Notification For Each Message

  • Customers gets notification for each message of the conversation in the same tab.

Chat widgets for each conversation

  • There are multiple chat widgets in the front-end as well as backend.
  • Website View
  • Backend View

Set the operators

The operators are required to activate the message channel for portal users.

Backend Chat View

Seller can chat through odoo default chat window.

Multiple Chat Support

Seller can chat with multiple buyers at a time.

Login to start chat

Start chatting from product page

User can also chat from seller profile page
